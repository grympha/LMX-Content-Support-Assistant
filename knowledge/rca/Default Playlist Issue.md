# Default Playlist Issue

## Issue

Default playlist unexpectedly displayed.

## Symptoms

- Scheduled content not visible
    
- Default playlist playing
    

## Investigation

Schedules reviewed.

Assignment issue discovered.

## Root Cause

Content not correctly assigned to target inventory.

## Resolution

Updated inventory assignment.

Redeployed content.

## Prevention

Verify:

- Inventory assignment
    
- Schedule assignment
    
- Deployment status
