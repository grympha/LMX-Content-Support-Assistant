# NovaStar TB60 Compatibility

## Issue

Attempted to deploy LMX Content on NovaStar TB60.

## Symptoms

- Application installed
    
- Pairing screen inaccessible
    
- Unable to control display
    

## Investigation

LMX Content launched behind NovaStar player interface.

## Root Cause

TB60 operates as a display controller and does not provide full third-party application control.

## Conclusion

NovaStar TB60 is not suitable for LMX Content deployment.

## Recommendation

Use supported Android or Windows devices.
