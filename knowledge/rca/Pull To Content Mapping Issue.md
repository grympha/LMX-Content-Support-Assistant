# Pull To Content Mapping Issue

## Issue

Pull To Content not functioning.

## Symptoms

- Inventory available
    
- Device online
    
- Content not displayed
    

## Investigation

Pairing successful.

Inventory available.

## Root Cause

Inventory mapping mismatch.

Inventory name did not match expected configuration.

## Resolution

Updated inventory mapping.

Redeployed content.

## Prevention

Verify:

- Inventory Name
    
- Device Mapping
    
- Pull To Content Configuration
