# Pull To Content Mapping Issue

## PTC Inventory Mapping Mismatch

A PTC inventory mapping mismatch occurs when the inventory name in the Pull To Content configuration does not match the expected device mapping. This causes content to not display even though the device is online and inventory is available.

## Issue

Pull To Content not functioning.

## Symptoms

- Inventory available
    
- Device online
    
- Content not displayed
    

## Investigation

Pairing successful.

Inventory available.

## Root Cause

Inventory mapping mismatch.

Inventory name did not match expected configuration.

## Resolution

Updated inventory mapping.

Redeployed content.

## Prevention

Verify:

- Inventory Name
    
- Device Mapping
    
- Pull To Content Configuration
