# Resolution Mismatch

## Content Cropped or Clipped on Screen

Content appears cropped on screen, content clipped at screen edges, or content stretched when the CMS layout resolution does not match the physical screen resolution.

## Symptoms

- Content cropped on screen or content clipped at edges
    
- Content stretched
    
- Black borders visible
    
- Layout not aligned
    

## Possible Causes

- Incorrect screen resolution
    
- Wrong content resolution
    
- Incorrect layout size
    
- LED processor scaling issue
    

## Basic Troubleshooting

1. Verify actual screen resolution.
    
2. Compare with CMS layout resolution.
    
3. Re-upload content using correct dimensions.
    
4. Confirm screen processor settings.
    
5. Redeploy content.
    

## Information Required

- Screen Resolution
    
- Layout Resolution
    
- Content Resolution
    
- Device Name
