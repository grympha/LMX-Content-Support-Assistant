# Device Offline

## Symptoms

- Device status shows Offline
    
- Unable to deploy content
    
- No recent heartbeat received
    

## Possible Causes

### Network

- Internet disconnected
    
- Firewall restriction
    
- DNS issue
    

### Device

- Device powered off
    
- Player application stopped
    
- System frozen
    

### CMS

- Device removed or replaced
    
- Incorrect pairing
    

## Basic Troubleshooting

1. Verify power connection.
    
2. Verify internet connection.
    
3. Check network cable or Wi-Fi.
    
4. Restart player application.
    
5. Reboot device.
    
6. Verify pairing status.
    

## Escalation Information

Collect:

- Device Name
    
- Device ID
    
- Last Online Timestamp
    
- Network Type
    
- Device Screenshot
