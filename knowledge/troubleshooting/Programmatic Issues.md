# Programmatic Issues

## Symptoms

- Programmatic content not playing
    
- VAST content fails
    
- Empty ad slots
    

## Requirements

### Android

- Android 11 or higher
    
- WebView 120 or higher
    
- Stable Internet Connection
    

### Windows

- Latest LMX Content Version
    

## Possible Causes

- Outdated WebView
    
- VAST URL inaccessible
    
- SSP integration issue
    
- Programmatic inventory configuration issue
    

## Basic Troubleshooting

1. Verify Android version.
    
2. Verify WebView version.
    
3. Verify internet connectivity.
    
4. Test VAST URL manually.
    
5. Verify inventory mapping.
    

## Escalation Information

Collect:

- Device Model
    
- Android Version
    
- WebView Version
    
- VAST URL
    
- Device Logs
