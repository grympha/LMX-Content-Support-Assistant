# VAST Issues

## Symptoms

- Application crashes during playback
    
- VAST content does not start
    
- Endless loading screen
    

## Requirements

- Android 11+
    
- WebView 120+
    
- Stable internet connection
    

## Common Root Causes

- Outdated Android System WebView
    
- Unsupported VAST version
    
- Invalid VAST URL
    
- Network timeout
    

## Basic Troubleshooting

1. Verify Android version.
    
2. Verify WebView version.
    
3. Open VAST URL in browser.
    
4. Clear application cache.
    
5. Restart application.
    

## Information Required

- Device Name
    
- Device Model
    
- Android Version
    
- WebView Version
    
- VAST URL
