# Application Force Close

## Symptoms

- Application closes automatically
    
- Player restarts repeatedly
    
- Content playback interrupted
    

## Possible Causes

- Low RAM
    
- Storage full
    
- Corrupted cache
    
- Unsupported content
    
- Application bug
    

## Basic Troubleshooting

1. Force stop application.
    
2. Clear cache.
    
3. Reboot device.
    
4. Verify available storage.
    
5. Reinstall application if necessary.
    

## Escalation Information

Collect:

- Device Model
    
- Application Version
    
- Android Version
    
- Crash Timestamp
