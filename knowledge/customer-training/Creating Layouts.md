# Creating Layouts

## Purpose

Layouts determine how content is displayed on the screen.

## Steps

1. Navigate to Layouts.
    
2. Create Layout.
    
3. Select screen resolution.
    
4. Add zones.
    
5. Save.
    

## Best Practices

- Match screen resolution.
    
- Test layout before deployment.
