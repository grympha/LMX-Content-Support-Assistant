# Creating Networks

## Purpose

Networks are used to organize screens into logical groups.

## Steps

1. Navigate to Networks.
    
2. Select Create Network.
    
3. Enter Network Name.
    
4. Save Network.
    

## Best Practices

- Use meaningful names.
    
- Separate clients into different networks.
    
- Avoid duplicate names.
