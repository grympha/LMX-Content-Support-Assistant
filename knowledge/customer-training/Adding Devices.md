# Adding Devices

## Purpose

Devices are responsible for content playback.

## Steps

1. Install LMX Content.
    
2. Launch application.
    
3. Record verification code.
    
4. Add device in CMS.
    
5. Complete pairing.
    

## Validation

- Device Online
    
- Device Assigned
    
- Content Visible
