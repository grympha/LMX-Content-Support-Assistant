# Creating Playlists

## Purpose

Playlists contain content that will be scheduled for playback.

## Steps

1. Navigate to Playlists.
    
2. Select Create Playlist.
    
3. Enter playlist name.
    
4. Add content.
    
5. Save.
    

## Supported Content

- Images
    
- Videos
    
- URLs
    
- HTML Packages
    
- Programmatic Content
