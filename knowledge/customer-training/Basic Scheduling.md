# Basic Scheduling

## Purpose

Basic Scheduling allows content to play within a defined date range.

## Steps

1. Create Playlist.
    
2. Create Layout.
    
3. Create Schedule.
    
4. Select dates.
    
5. Assign locations.
    
6. Save.
    

## Validation

- Schedule active
    
- Device online
    
- Content deployed
