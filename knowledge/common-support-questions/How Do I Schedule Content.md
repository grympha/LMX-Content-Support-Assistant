# How Do I Schedule Content?

## Steps

1. Create Playlist.
    
2. Create Layout.
    
3. Upload Content.
    
4. Create Schedule.
    
5. Select Date Range.
    
6. Assign Locations.
    
7. Save Schedule.
    
8. Deploy Content.
    

## Best Practices

- Verify dates.
    
- Verify locations.
    
- Verify playlist assignment.
    
- Confirm device online status.
