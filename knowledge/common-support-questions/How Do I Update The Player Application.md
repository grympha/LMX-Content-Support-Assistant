# How Do I Update The Player Application?

## Windows

1. Download latest installer.
    
2. Run installer.
    
3. Follow installation instructions.
    
4. Restart application.
    

## Android

1. Download latest APK.
    
2. Install APK.
    
3. Launch application.
    
4. Verify version.
    

## Verification

Confirm:

- Application starts successfully.
    
- Device remains paired.
    
- Content plays normally.
