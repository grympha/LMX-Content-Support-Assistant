# What Formats Are Supported?

## Supported Images

- JPG
    
- JPEG
    
- PNG
    

## Supported Videos

- MP4 (Recommended)
    
- MOV
    

## Supported Interactive Content

- URL
    
- HTML Package (ZIP)
    

## Supported Programmatic Content

- VAST
    

## Recommendation

Use MP4 (H.264 codec) for best compatibility.
