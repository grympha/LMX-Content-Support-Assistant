# How To Ask Good Support Questions

## Why Is This Important?

Providing complete information helps resolve issues faster.

## Information To Provide

### Device Information

- Device Name
    
- Device ID
    
- Player Version
    
- Operating System
    

### Issue Details

- What happened?
    
- When did it happen?
    
- Is the issue reproducible?
    

### Evidence

- Screenshot
    
- Video recording
    
- Error message
    

## Example

Poor Question:

"Content not working."

Good Question:

"Device KL-Mall-01 is online but scheduled content is not playing since 10:00 AM. Screenshot attached."
