# Programmatic Content

## What is Programmatic Content?

Programmatic Content allows ads to be delivered dynamically through advertising platforms.

## Requirements

### Android

- Android 11+
    
- WebView 120+
    

### Windows

- Latest LMX Content Version
    

## Supported Integrations

- SSP
    
- DSP
    
- VAST
    

## Common Issues

- WebView outdated
    
- Inventory mapping issue
    
- VAST errors
