# Layouts

## What is a Layout?

A Layout defines how content appears on the screen.

Layouts may contain:

- Single Zone
    
- Multiple Zones
    
- Video Zones
    
- Image Zones
    
- Widget Zones
    

## Common Layout Types

### Full Screen

Single content zone.

### Split Screen

Multiple content zones displayed simultaneously.

## Best Practices

- Match layout resolution to screen resolution.
    
- Verify content placement.
    
- Test before deployment.
