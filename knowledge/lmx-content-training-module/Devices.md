# Devices

## What is a Device?

A Device is the media player running LMX Content.

Examples:

- Windows PC
    
- Android Box
    
- Android Tablet
    

## Device Functions

- Download content
    
- Play content
    
- Send status updates
    
- Generate playlogs
    

## Device Status

### Online

Device is connected and communicating.

### Offline

Device is not communicating.

## Best Practices

- Use meaningful device names.
    
- Verify screen resolution.
    
- Monitor storage availability.
