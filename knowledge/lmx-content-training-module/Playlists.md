# Playlists

## What is a Playlist?

A Playlist contains one or more content items.

Playlists are assigned to schedules for playback.

## Supported Content Types

- Images
    
- Videos
    
- URLs
    
- HTML Packages
    
- VAST Content
    

## Best Practices

- Use descriptive playlist names.
    
- Group similar content together.
    
- Review playlist content before deployment.
