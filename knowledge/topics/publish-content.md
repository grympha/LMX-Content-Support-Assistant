# Publish Content

Publishing sends approved schedules and playlist changes to the relevant devices.

## Steps

1. Confirm the schedule setup is complete.
2. Confirm Default Playlist has at least one scheduled image and one scheduled video.
3. Confirm content uses valid formats such as MP4, JPG, or PNG.
4. Save content and schedule changes.
5. Approve content where required.
6. Click Publish.
7. Wait for device sync.
8. Validate playback on the device.

## Important Notes

- A common publish error means Default Playlist does not have required scheduled image/video content.
- Uploaded content must be scheduled, not only stored.
- Device sync may take time.

## Common Mistakes

- Default Playlist missing image or video.
- Unsupported file type.
- Schedule saved but not approved.
- Wrong target playlist.
