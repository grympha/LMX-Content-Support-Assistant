# Playlist Creation
## Overview
A Playlist in MW Content is a sequence of media files (videos, images, etc.) that plays on screens assigned to a location. It determines the content order, duration, and timing. Playlists are essential for scheduling content in a controlled loop on any signage screen.
## How to Create a Playlist (Step-by-Step)
- Go to Setup → Playlist
- Click Create Playlist
- Fill in:
Playlist Name
Duration (Seconds)
Select Type (Normal, Fixed, Prime, Dependent)
Assign a Location
Leave Play Type as Timer
(Optional) Add Tags, Description, Media Partner
- Ensure Status is checked
- Click Save
## Fields Explained
Field
Required
Description
Play List Name
✅
Unique name to identify the playlist (e.g., Retail_Loop_AM)
Duration (Seconds)
✅
Total loop time in seconds (e.g., 60, 120)

❌
Choose the playlist type based on scheduling needs (see details below)
Location
✅

Play Type

Keywords for filtering or searching playlists
Description
❌
Notes or purpose of the playlist (e.g., “Main product ads”)
Media Partner
❌
Enable if the playlist is for external media partners
Display Timer
❌
Show countdown timer on the player for content switching
Status
✅
Keep checked to activate playlist for use
## Playlist Types Explained
Type
Use Case
Normal (default)
Manual scheduling; you decide content order, duration
Fixed
Duration-based loop; content automatically fills allocated time
Prime
Runs content only during peak times (e.g., lunch hours)
Dependent
Paired with another playlist; shows content related to a primary playlist
## Best Practices
- Set playlist duration based on total content length (e.g., 60 seconds = 3 x 20s ads)
- Use Prime Time for scheduled day-part campaigns
- Always assign a Default Playlist for backup in case main content is unavailable
- Use Tags for grouping playlists by campaign or region
