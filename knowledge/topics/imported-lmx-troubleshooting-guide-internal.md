# LMX Troubleshooting Guide (Internal)
A Basic Troubleshooting Guide for CMS Platform
Versions of these steps maintained here ons
## Basic Steps
- Check Internet Connection
- Ensure the CMS platform is connected to the internet.
- Verify that the network is stable by testing with other devices or websites.
- Restart CMS Application
- Close and restart the CMS platform to clear any temporary issues.
- Verify Software Updates
- Ensure the CMS platform is running the latest version.
- Check for any pending updates and install them.
- Check for Errors in Logs
- Review CMS logs for any error messages or warnings.
- Identify specific error codes and investigate them.
- Verify Content and Playlists
- Ensure content (images, videos, etc.) is properly uploaded and not corrupted.
- Check that playlists and schedules are correctly assigned and active.
- Test with Simple Content
- Test the platform by uploading and displaying a basic media file (e.g., an image or video).
- Clear Cache and Temp Files
- Clear the CMS cache and temporary files to improve performance and resolve issues.
- Check Database Health
- Ensure the CMS database is functioning properly and free from corruption.
- Run any necessary database repairs or optimizations.
- Restart the CMS Server (if applicable)
- If the CMS is hosted on a server, restart the server to resolve any backend issues.
