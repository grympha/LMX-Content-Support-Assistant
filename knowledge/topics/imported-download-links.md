# LMX Content – Download Links

## Overview

The following are the official download links for the MW/LMX Content player application across all supported platforms. Always install the latest version directly over the existing version without uninstalling first.

## Latest Versions

| Platform | Version |
|---|---|
| Windows | V1.0.38 |
| Linux / Ubuntu | V1.0.35 |
| Android | V2.9.1.2 Native |
| BrightSign | V1.1.3 |

## Download Links

### Windows – V1.0.38

Download the Windows installer (.exe):

https://mw-lmx-content-hosting.s3.ap-south-1.amazonaws.com/player-builds/updates/windows/1.0.38/MW+Content-setup-1.0.38.exe

Installation steps:
1. Download the installer.
2. Run MW Content-setup-1.0.38.exe.
3. Complete the installation wizard.
4. Launch the player.

### Linux / Ubuntu – V1.0.35

Download the Linux AppImage:

https://mw-lmx-content-hosting.s3.ap-south-1.amazonaws.com/player-builds/updates/linux/1.0.35/mac-media-player-1.0.35.AppImage

Installation steps:
1. Download the AppImage file.
2. Make the file executable: chmod +x mac-media-player-1.0.35.AppImage
3. Run the AppImage.
4. Launch the player.

### Android – V2.9.1.2 Native

Download the Android APK:

https://s3.ap-south-1.amazonaws.com/player.movingwalls.com/android_app/apps/Android/webview/2.9.1.2/lmx_web_28_1_291.apk

Installation steps:
1. Download the APK file.
2. Enable installation from unknown sources on the device.
3. Install the APK.
4. Do NOT uninstall the existing version before installing.
5. Launch the player.

### BrightSign – V1.1.3

Download the BrightSign package (.zip):

https://moving-walls-live.s3.ap-southeast-1.amazonaws.com/player-builds/updates/brightsign/update/LMX-Player.zip

Installation steps:
1. Download the ZIP file.
2. Extract the package.
3. Follow the BrightSign deployment process.
4. Launch the player.

## Important Notes

- Always install over the existing version – do not uninstall first.
- Installing over the existing version preserves device pairing, configuration, and cached content.
- Verify the device is online after installation.
- Generate a new verification code only if pairing was lost.
- For Android, ensure installation from unknown sources is enabled before installing the APK.

## Supported Platforms

- Windows (x64)
- Linux / Ubuntu (AppImage)
- Android (Native WebView APK)
