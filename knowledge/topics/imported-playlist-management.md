# Playlist Management
## Overview
After creating a playlist in MW Content, you’ll need to manage and schedule its content. The Playlist Management module allows you to:
- Add, remove, or reorder content
- Approve and call off content
- Set up daily or weekly scheduling
- Preview playlists
- Track status and duration conflicts
This module ensures content is organized, scheduled correctly, and ready for playback across assigned devices.
## Playlist Management Functions
Feature
Purpose

Reorder Content

Schedule Content
Set start/end dates, repeat rules, and playback days
Call Off Content
Temporarily deactivate media from the playlist
Preview Playlist
Visual preview of how the playlist appears on screen
Approve Content
Confirm scheduled content for final playback
Clone Playlist
Create a duplicate with the same settings
Duration Check
Alert if total duration exceeds or underutilizes set loop time
## How to Manage a Playlist (Step-by-Step)
- Navigate to:
Scheduled Content Tab → Select Network → Location → Playlist
- Click Edit or View Content
- From here you can:
Add media from Storage
Set scheduling options:
Start/End Date
Days of the Week
Repeat Count
Schedule Type (Client, Mall, Filler, etc.)
Use Call-Off to deactivate content temporarily without deleting it
Drag items to reorder
Click Preview to simulate the playlist
Click Approve to finalize the scheduled playlist

## Approval Workflow
- New content added to a playlist remains in Pending state
- Admin or approver must review and approve it before it’s live
- This ensures quality control before broadcasting
## Best Practices
- Maintain a playlist duration that matches the screen loop (e.g., 120 seconds)
- Use tags for easy playlist search and filtering
- Always preview playlists after major updates
- Keep a default playlist assigned as a fallback
- Group content by campaign or message type for clarity
