# Dashboard Overview

Use the Dashboard as the first page to understand the current CMS and device status.

## Steps

1. Log in to LMX Content CMS.
2. Open the Dashboard landing page.
3. Review Online Devices, Offline Devices, and Unpaired Devices.
4. Check Storage Usage to understand available media capacity.
5. Open Notifications for pending content, placeholder content, and instant events.
6. Review Release Information for CMS platform and player build versions.

## Important Notes

- Offline devices should be investigated from Device Manager.
- Unpaired devices usually appear after fresh installs or factory resets.
- Storage usage should be checked before uploading large files.
- Placeholder or pending content should be resolved before publishing campaigns.

## Common Mistakes

- Ignoring offline or unpaired device counters.
- Uploading content without checking storage capacity.
- Missing release/player version information during training or setup review.
