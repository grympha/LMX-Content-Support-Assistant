# Recommended Android Devices for MW Content
### Minimum Requirements
- OS: Android 11.0 and above
- CPU: Rockchip RK3328 / Amlogic S905 (Cortex A53 Quad-Core)
- System Type: 64-bit OS (recommended)
- Memory:
- Recommended: 8 GB RAM / 128 GB ROM
- Minimum: 4 GB RAM / 64 GB ROM
