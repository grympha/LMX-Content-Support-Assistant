# Recommended Windows Devices for LMX Content
### LMX Content Requirement for Windows
- OS: Windows 10 and 11
- Processor:
- Recommended: Intel Core i5 or i7
- Minimum: Intel Pentium II, III or AMD Processor
- System Type: 64-bit (recommended)
- RAM / Graphics:
- Recommended: 8 GB RAM or more
- Minimum: 4 GB RAM + 1 GB Graphics Card
### Key Things to Check
- Ensure Windows is fully updated (latest updates, drivers, codecs).
- Devices should support hardware-accelerated video decoding (H.264, H.265) for smooth media playback.
- Opt for fanless or low-noise models if the device will be used in public or quiet environments.
- Devices with dual HDMI or DisplayPort output are ideal for dual-screen setups.
