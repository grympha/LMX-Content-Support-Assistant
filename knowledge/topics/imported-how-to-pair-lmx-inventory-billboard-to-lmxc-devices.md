# How to pair LMX inventory/billboard to LMXC Devices
Prerequisites
- Make sure LMX Integration has been enabled for the tenant.
- You’re logged in as a Tenant Admin.
Steps
- Log in to LMX Content (LMXC) as Tenant Admin.
- Go to Devices Manager (Devices → Device List).
- Select the device → click Edit.
- Under LMX Intergration tab, in the Billboard field, search and select the billboard to pair with this device.
- Select the playlist that should receive the scheduling that goes through that Inventory from the SSP. You can select one or multiple.
- Enter Sync Interval (minutes)(if shown for device; otherwise it inherits the tenant setting).
- Click Update/Save for changes to take effect.
