# Recommended BrightSign Devices for MW Content
### MW Content Requirement for BrightSign
- OS: BrightSign OS version 6.2
- Processor: ARM Cortex-A15, Quad-Core 1.0–2.0 GHz
- System Type: 64-bit (recommended)
- RAM / Storage:
- Recommended: 4 GB RAM / 16 GB ROM
- Minimum: 2 GB RAM / 8 GB ROM
- Tested Compatibility: XT1143, HD224, HS123
### Notes
- All models must be updated to OS version 6.2 or later for full MW Content compatibility.
- MW Content has been successfully tested on XT1143 and HD224 with stable results.
- When deploying multiple units, use BrightAuthor:connected or BSN.cloud for device and content management.

- For high-end signage (dual 4K, complex HTML5, multiple zones):
→ XT1143

→ LS425
