# Storage Management

Storage is used to upload, organize, manage, and clean up media files.

## Steps

1. Open the Storage section.
2. Check available storage capacity.
3. Upload supported media files.
4. Organize files into clear folders.
5. Use uploaded assets in playlists or schedules.
6. Remove unused content carefully when storage is near the limit.

## Important Notes

- Storage usage appears on the Dashboard.
- Deleting content can affect playlists or schedules that use it.
- Supported formats should be checked before scheduling.

## Common Mistakes

- Uploading files without organizing folders.
- Deleting content that is still scheduled.
- Ignoring storage capacity before large uploads.
