# Bundle Scheduling

Bundle scheduling helps manage grouped schedules or multiple related content items together.

## Steps

1. Confirm target Network and Location.
2. Prepare playlist and content assets.
3. Create or select the bundle schedule.
4. Set active dates.
5. Set timing and playback rules.
6. Assign the correct playlist.
7. Save and approve where required.
8. Publish the bundle schedule.
9. Validate playback and metrics.

## Important Notes

- Bundle schedules still depend on correct playlist targeting.
- Default Playlist requirements still apply.
- Devices must sync after publish.

## Common Mistakes

- Wrong target playlist.
- Missing approval.
- Schedule timing outside playback hours.
- Publishing before default content is ready.
