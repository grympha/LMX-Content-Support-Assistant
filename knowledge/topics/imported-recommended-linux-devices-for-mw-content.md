# Recommended Linux Devices for MW Content
### MW Content Requirement for Linux
- OS: Ubuntu 18.04 LTS and above
- Processor:
- Recommended: Intel Core i5, i7
- Minimum: Intel Pentium II, III
- System Type: 64-bit (recommended)
- RAM / Graphics:
- Recommended: 8 GB RAM or above
- Minimum: 4 GB RAM + 1 GB Graphics Card
### Recommendation
For professional-grade signage and compatibility with MW Content:
- Stick with Intel NUC, Lenovo Tiny, or ASUS PN series.
- Make sure the device has hardware-accelerated video support and a dedicated GPU or strong iGPU (e.g., Intel Iris Xe).
