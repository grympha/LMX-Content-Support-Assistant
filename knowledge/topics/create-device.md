# Create Device

A Device is the CMS record for the physical screen or player that displays scheduled content.

## Steps

1. Go to Device Manager.
2. Click Add Device.
3. Enter Device Name.
4. Assign the device to a Location.
5. Confirm the Network selected from the location.
6. Set Start Time and End Time.
7. Select Screen Type.
8. Keep Status enabled.
9. Save the device.
10. Copy the generated Verification Code.

## Important Notes

- The device must be assigned to the correct location.
- The verification code is used during player app registration.
- Verification code is one-time use only.

## Common Mistakes

- Wrong location assignment.
- Reusing an already-used verification code.
- Non-unique device name.
- Status disabled.
