# Default Playlist

Default Playlist is mandatory fallback content and is required before scheduling and publishing workflows.

## Steps

1. Go to Setup > Default Playlist.
2. Click Create or add content.
3. Select one image creative.
4. Create schedule for the image.
5. Set duration, start/end date, daypart or spot.
6. Save the image schedule.
7. Approve the image content.
8. Select one video creative.
9. Create schedule for the video.
10. Set duration, start/end date, daypart or spot.
11. Save the video schedule.
12. Approve the video content.

## Important Notes

- Default Playlist must have at least one scheduled image and one scheduled video.
- Uploading content alone is not enough.
- Content must be scheduled, saved, and approved.
- Publishing can be blocked if Default Playlist requirements are not met.

## Common Mistakes

- Only uploading default content without scheduling it.
- Missing either image or video schedule.
- Forgetting approval.
- Using unsupported formats.
