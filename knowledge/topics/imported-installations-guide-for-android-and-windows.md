# Installations Guide for Android and Windows
## Important Notes
- To update the existing application:
Please download and install the new version without uninstalling the current or previous version.
- For new device installation:
Please create the device in the CMS platform to generate a verification code.
Click here for the installation guide
Note: SR = Software Render build — designed for devices without GPU acceleration, relying on CPU-based rendering instead.
To setup and install a new player, please follow the guides below
## Android Version
Step 1: After download the app click install.
Step 2: Click “Install”
Step 3: Allow the permission 1/2 and 2/2
## For Registration
Step 1: Select “Login with Verification Code”
Step 2: Use the Verification Code create in the MW Content platform
Note: Verification code are One time use only
## Windows Version
Step 1: After download the installer Right click and Click “Run as Administration”
Step 2: On windows protection Pop-out select “Run anyway”
Step 3: Registration - Use the Verification Code that Are create in the platform
Note: Verification code are One time use only
