# Schedule Content

Scheduling content assigns media or campaigns to the correct playlist, location, date, time, and playback rule.

## Steps

1. Confirm Network is created.
2. Confirm Location is created with the correct timezone.
3. Confirm Playlist exists and is assigned to the correct location.
4. Confirm Default Playlist has at least one scheduled image and one scheduled video.
5. Upload or select supported content.
6. Create the schedule.
7. Set duration.
8. Set start date and end date.
9. Configure daypart or spot if required.
10. Save the schedule.
11. Approve content if required.
12. Publish to the relevant playlist or devices.
13. Wait for device sync.
14. Validate playback.

## Important Notes

- Timezone must match the location.
- Default Playlist must be ready before publishing.
- Device sync can take time after publishing.
- Use valid formats such as MP4, JPG, and PNG.

## Common Mistakes

- Wrong playlist or location.
- Schedule outside active playback time.
- Content saved but not approved.
- Publishing before Default Playlist is prepared.
