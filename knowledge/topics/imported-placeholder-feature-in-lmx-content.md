# Placeholder Feature in LMX Content
The Placeholder option in allows users to temporarily fill available slots in a playlist with substitute content. This is especially useful when the final media assets are not yet ready, enabling proactive slot booking and seamless replacement.
## Feature Overview
### Enable Placeholder
- When enabled, users can schedule temporary content for up to 3 days.
- An email registration alert will be triggered, prompting users to input a valid mail ID.
- This ensures timely reminders and smooth content replacement with finalized media.
### Visual Highlight
- Clicking on a placeholder entry will highlight the corresponding slot in the Schedule Content page.
- This helps users quickly identify and manage placeholder content in the schedule.
## Use Case Example
A media owner has a playlist loop duration of 120 seconds:
- 100 seconds are already occupied by confirmed content.
- 20 seconds remain unfilled and are temporarily assigned placeholder content.
- When a client provides final content for the remaining 20 seconds, the placeholder can be easily replaced without disrupting the playlist.
## Content Replacement Rules
- Placeholder content can only be replaced by content of the same duration.
- If the new content exceeds the original duration, it must be manually adjusted to fit the loop.
## Benefits
- Enables early slot reservation while awaiting final assets.
- Simplifies the content update process for short-notice campaigns.
- Improves operational efficiency by clearly marking and managing temporary content.
