# Create Location

A Location represents where screens are deployed, such as a building, branch, floor, zone, or screen group.

## Steps

1. Go to Setup > Location.
2. Click Create.
3. Enter Location Name.
4. Select Country.
5. Select Region.
6. Select the correct Time Zone.
7. Enter Zone Name.
8. Enter Subzone Name.
9. Add Tags and Description if useful.
10. Keep Status enabled.
11. Save the location.

## Important Notes

- Time Zone is important because it affects scheduled content playback time.
- Locations are linked under Networks.
- Use clear naming such as COUNTRY_REGION_ZONE_SUBZONE.

## Common Mistakes

- Selecting the wrong timezone.
- Creating devices before the location structure is ready.
- Using unclear location names.
