# Enable Multi-Factor Authentication (MFA) – Tenant Level
### Step-by-Step Guide
#### Step 1: Enable MFA for the Tenant
- Login to the LMX Dashboard
- Go to My Account > Edit
- Enable the MFA toggle
- Choose Google Authenticator or Email Authentication
- Click Update
You will be logged out and redirected to the login screen.
### Google Authentication Setup
#### Step 2: Install the Authenticator App
- Download the “Authenticator” app on your phone or PC
- Chrome Extension
#### Step 3: Scan or Enter the Security Code
- Open the app, scan the QR code or enter the manual security code
- Optionally: click Edit → + (plus icon) → Manual entry
- Enter:
- Secret Code from LMX
- Issuer (e.g., your username)
#### Step 4: Enter Verification Code
- Copy the 6-digit code from the app
- Paste it into the text box on LMX and click Continue
- MFA is now enabled
### Step 5: Save Recovery Codes
- Click Download to save a file containing 10 one-time recovery codes
- Click Done
### Email Authentication (Alternative Option)
#### Step 2: Choose “Add Email”
- Click Continue
- A 6-digit verification code will be sent to your email
- Wait a few seconds and copy the code
#### Step 3: Enter Verification Code
- Paste the code into the text box
- Click Continue
- Email MFA is now enabled
### Login Experience After MFA Setup
- Upon future logins, users must:
- Enter username and password
- Then provide the 6-digit code via:
- Google Authenticator, or
- Email (OTP valid for 10 minutes)
- You may also use recovery codes if needed.
