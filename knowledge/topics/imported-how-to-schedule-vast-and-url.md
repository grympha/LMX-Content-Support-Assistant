# How to Schedule Vast and URL
Step 1:
Select “Schedule Content” on the dashboard.
Step 2:
Select Network and Location.
Step 3:
Select “create” in the right conner
Step 4:
Select Widget.
Step 5:
Select “Vast” or “URL”
URL:
- Fill the URL name that will appear in the Schedule content dashboard.
- Copy and Paste the URL link in the URL box.
- Save and Select for content approval
Vast:
- Select “Filler Content”
Filler Content will play if the vast link not trigger
- Fill out the name of the content.
- Select Environment
1. Live Environment
- fill the LMX Id Deal
- fill the LMX Inventory Id (Reference ID/UUID)
2 . Custom Environment
- Fill the Vast Tag Url
- Save the configurations.
