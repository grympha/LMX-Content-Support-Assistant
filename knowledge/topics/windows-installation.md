# Windows Installation

Windows installation sets up the MW Content player and registers it to CMS using a verification code.

## Steps

1. Create the device in CMS first.
2. Download the Windows installer.
3. Right click the installer.
4. Select Run as Administrator.
5. Choose Run Anyway if Windows protection appears.
6. Enter the CMS verification code during registration.
7. Confirm the player starts syncing.

## Important Notes

- Windows 10 and Windows 11 are supported.
- Keep Windows updated with drivers and codecs.
- Verification codes are one-time use only.

## Common Mistakes

- Not running installer as administrator.
- Using the wrong verification code.
- Missing Windows updates, drivers, or codecs.
