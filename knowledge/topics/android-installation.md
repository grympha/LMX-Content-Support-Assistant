# Android Installation

Android installation prepares the MW Content player app and pairs it to CMS using a one-time verification code.

## Steps

1. Create the device in CMS first.
2. Copy the generated verification code.
3. Download the MW Content Android app.
4. Install the app.
5. Allow required permissions.
6. Choose Login with Verification Code.
7. Enter the verification code.
8. Confirm the device starts syncing.

## Important Notes

- Install updates without uninstalling the previous app unless instructed.
- Verification codes are one-time use only.
- Android 11 and above is recommended.
- Device needs internet access.

## Common Mistakes

- Using an expired verification code.
- Skipping permissions.
- Pairing to the wrong device record.
- Device has no internet connection.
