# Create Network

A Network is the primary structure used to organize devices, users, content rules, playback hours, and reporting behavior.

## Steps

1. Go to the Network section in CMS.
2. Click Create.
3. Enter Network Name.
4. Add Description and Tags if useful.
5. Set Start Time and End Time for default playback hours.
6. Configure Playlog Interval.
7. Enable Play Log for reporting.
8. Enable Background Download if devices should cache content.
9. Configure Volume and Brightness if needed.
10. Keep Status enabled.
11. Save the network.

## Important Notes

- Create separate networks for different clients, regions, permissions, or schedule rules.
- Playlog Interval controls how often the device checks in to upload played-content data.
- Keep Play Log enabled for reports and analytics.

## Common Mistakes

- Using one network for unrelated clients or regions.
- Setting the wrong start/end playback hours.
- Disabling Play Log.
- Forgetting to enable Status.
