# Android APK Troubleshooting Steps for LMX Content
If you're facing issues with the LMX Content APK on an Android device, follow these troubleshooting steps:
NOTE: Do Not Clear Storage
❌ Why? – Clearing storage will cause the device to lose its sync with the platform.
🔹 Impact: If storage is cleared, the device will require re-registration on the platform.
🔹 Issue: The verification code provided by the platform is one-time use only, meaning a new device have to create in order to sync with the device again.
### 1. Basic Checks
✅ Ensure the device is powered on and connected to the internet (Wi-Fi or mobile data).
✅ Verify that the APK is installed correctly and updated to the latest version.
✅ Restart the device to refresh system processes.
### 2. Application-Specific Issues
## 🔹 App Not Opening or Crashing
- Clear cache & data:
1️⃣ Clear Cache & Data:
- Go to Settings → Apps → LMX Content → Storage → Clear Cache.
2️⃣ Check Storage Space:
- Ensure the device has enough available storage. Low storage can cause crashes.
3️⃣ Restart the Device:
- A simple restart can resolve temporary software glitches.
4️⃣ Uninstall and Reinstall the APK (Only if the above steps don’t work):
- If the issue persists, uninstall the app and reinstall the latest version.
## 🔹 Content Not Loading or Playing
- Check if the device is online by opening a web browser.
- Ensure correct content scheduling in LMX.
1️⃣ Verify Content Resolution & Format:
- Ensure the uploaded content matches the screen resolution and supported file format. (Incorrect resolution may cause display issues.)
2️⃣ Check Scheduling Time & Date:
- Confirm the scheduled time aligns with the intended playback window. (Mismatched scheduling can result in content not playing.)
3️⃣ Avoid Overloading Playlists:
- Do not add excessive content to a single playlist. (Too many files can cause syncing failures.)
4️⃣ Confirm Network Connectivity:
- Make sure the device is connected to a stable network. (Poor connectivity can delay content updates.)
- Restart the app and reload the content.
## 🔹 Screen Stuck or Frozen
- Force-stop the app:
- Settings → Apps → LMX Content → Force Stop, then restart the app.
- Restart the device.
### 3. Network & Server Issues
🔹 Content Not Updating
- Verify network connectivity (try another Wi-Fi or hotspot).
- Restart the router or modem if needed.
- Check for any server outages in LMX.
🔹 Slow Performance or Buffering
- Reduce the number of apps running in the background.
- Go to Settings → Apps → Running Apps (or App Info in some versions).
Using Recent Apps Button (Easiest & Quickest)
- Tap the Recent Apps button (square or three-line icon at the bottom) → Swipe left or right to browse running apps → Swipe up (or tap Close All) to stop background apps.
- Check the list of active apps and force-stop unnecessary ones.
- Check available storage and free up space if necessary.
### How to Check Available Storage and Free Up Space on Android
📌 Step 1: Check Storage Usage
- Open Settings on your device.
- Scroll down and select Storage (or Device Care → Storage on some devices).
- Review the breakdown of used space (Apps, Photos, Videos, etc.).
📌 Step 2: Identify Large Apps
- Go to Settings → Apps (or Application Manager).
- Tap Sort by Size (if available) to see which apps use the most space.
- Select any unnecessary app and tap Uninstall or Clear Cache.
📌 Step 3: Remove Unused Files & Media
- Open File Manager or Files app.
- Navigate to Downloads, Videos, or Large Files.
- Delete old or unnecessary files to free up space.
📌 Step 4: Use Storage Cleanup Tools (Optional)
- Some Android devices have built-in "Free Up Space" or "Smart Storage" options.
- Go to Settings → Storage → Free Up Space and follow the recommendations.
- Try a different network connection.
### 4. Device Compatibility & Settings
🔹 Ensure Device Compatibility
- Check if the device meets the minimum system requirements for LMX Content.
- Ensure Android OS is updated to the latest supported version.
🔹 Check Permissions
- Go to Settings → Apps → LMX Content → Permissions and ensure all required permissions are enabled.
### 5. Reinstallation & Logs
🔹 Reinstall the APK
- Uninstall the app completely and reinstall a fresh copy.
- Ensure the APK is downloaded from a trusted source.
🔹 Check Logs for Errors
- If possible, access system logs via ADB or a logging tool to identify error messages.
### 6. Contact Support
If the issue persists, collect the following details and contact support:
✅ Device model and Android version
✅ APK version
✅ Screenshot or video of the issue
✅ Error messages (if any)
