# Step by step to replace devices in LMX Content
## Most Common Reasons to Replace Devices in LMX Content
- Hardware issues: Device is faulty, unresponsive, or has playback/display problems.
- Device upgrade: Replacing with newer hardware for better performance or resolution.
- Relocation: Device is being reassigned to a different site or replaced at the same location.
- Security & compliance: Old device no longer meets current system or security standards.
- Operational cleanup: Standardizing naming, grouping, or removing improperly configured devices.
- New screen onboarding: Swapping devices without changing campaign or content assignments.
## Pre-Replacement Checklist for Devices in LMX Content
- Record Current Device Info
- Device Name, Group, Network, Campaign, Location.
- Active playlists or scheduled campaigns.
- Unlink or Deactivate Old Device
- Set the status to Inactive or Unassigned.
- Ensure it’s removed from campaign targeting if necessary.
- Prepare the New Device
- Ensure the device is powered, connected to internet, and has the LMX Player App installed.
- Get the verification code from platform.
- Check Network & Location
- Confirm the new device is being placed in the correct location and network group.
- Update any site info or GPS coordinates if needed.
- Match Content Assignment
- Ensure the new device is ready to receive the same campaigns, playlists, and scheduling as the old one.
## Step by Step Instructions
#### Step 1: Download Playlogs from Previous Device
- Go to Devices Manager in the LMX Content platform.
- Select the old device.
- Navigate to the Playlogs or Logs section.
- Export or download all historical playlogs for audit or record-keeping.
You can follow the guides here.
#### Step 2: Create New Device in LMX Content
- In the LMX Content portal, go to Devices > Add Device.
- Fill in:
- Device Name
- Network & Location
- Set screen time and resolution.
You can follow the guides here.
#### Step 3: Install LMX Content App on New Device
- Download and install the LMX Content Player App on the new Android device.
- Make sure the device is connected to the internet.
You can follow the guides here.
#### Step 4: Key In Verification Code
- Launch the LMX app on the new device.
- A key-in verification code will be displayed.
- In the CMS, go to Device Manager, select the newly created device and copy the verification code.
#### Step 5: Validate All Content
- Assign the correct playlist or campaign to the new device.
- Make sure:
- All media assets are downloaded successfully.
- Content starts playing correctly.
- Check in the CMS under Device Monitoring for:
- Online status
- Last heartbeat
- Current playing file
