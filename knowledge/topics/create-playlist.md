# Create Playlist

A Playlist is a sequence of media files that plays on screens assigned to a location.

## Steps

1. Go to Setup > Playlist.
2. Click Create Playlist.
3. Enter Playlist Name.
4. Set Duration in seconds.
5. Select playlist Type: Normal, Fixed, Prime, or Dependent.
6. Assign the playlist to a Location.
7. Keep Play Type as Timer unless another trigger is required.
8. Add Tags, Description, or Media Partner if needed.
9. Keep Status enabled.
10. Save the playlist.

## Important Notes

- Normal playlist is used for manual scheduling.
- Prime playlist is useful for peak-time or day-part campaigns.
- Playlist duration should match the intended content loop.
- Always assign the playlist to the correct location.

## Common Mistakes

- Playlist assigned to the wrong location.
- Playlist duration does not match content duration.
- Status disabled.
- No default playlist prepared as backup.
