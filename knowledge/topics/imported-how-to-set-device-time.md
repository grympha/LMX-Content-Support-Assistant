# How to Set Device Time
Device Time in the CMS to enable automatic device startup and shutdown. Setting the correct device time helps ensure screens power on and off at scheduled hours, improving stability and reducing the need for manual intervention.
## For New Device Creation:
- Navigate to Device Manager.
- Click Add Device.
- In the creation form, go to the Screen/User section.
- Locate the Device Time settings.
- Set the desired Start Time and Close Time for the device.
- Save the configuration.
## For Existing Devices:
- Go to Device Manager.
- Locate the device you want to update and click the Edit icon under the Action column.
- Find the Screen/User section in the edit device.
- Update the Device Time with your preferred startup and shutdown time.
- Save your changes.
