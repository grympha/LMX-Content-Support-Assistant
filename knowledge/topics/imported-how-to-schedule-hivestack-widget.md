# How To Schedule Hivestack Widget
- Log in to the CMS – Go to https://cms.movingwalls.com/ and enter your credentials.
- Go to the Storage Page or “Create” section if the user is in the “Scheduled Content” page.
- Go to the Widget Tab – Click on the “Widgets” tab located in the top right corner to select preffered widget.
- Select the “Hivestack SSP” Widget and click “Create Schedule” button on the bottom right.
## Configuration Page
- Select the environment of Ad Serving.
- Add the duration of Ad Slot
- Add a filler content which will play as fallback if ad serving fails (due to no internet, no response, wrong configuration)
- Add an organization ID “API Key” that is given from the Hivestack team.
- Lastly, add the “Screen ID” as the Ad unit ID that was onboarded with Hivestack. (You can check the “use device ID as Screen ID” box if Hivestack uses our CMS device ID).
- Finally you can click “Save” to publish it to the relevant playlists.
