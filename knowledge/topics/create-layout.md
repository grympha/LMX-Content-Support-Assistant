# Create Layout

A Layout divides a screen into zones, called tags, with each tag linked to a playlist.

## Steps

1. Go to Setup > Layout.
2. Click Create Layout.
3. Enter Layout Name.
4. Assign the layout to a Location.
5. Choose Layout Scale Type: Percentage or Pixel.
6. Select Screen Type or resolution.
7. Add Tags.
8. For each tag, set name, playlist, left/top position, width, and height.
9. Save the layout.

## Important Notes

- Every tag must be linked to a playlist.
- Percentage scaling is better for screens of different sizes.
- Pixel scaling is better for fixed, known screen resolutions.
- Keep layouts simple and readable.

## Common Mistakes

- Creating tags without playlists.
- Adding too many zones.
- Selecting the wrong resolution.
- Forgetting to set a default layout where required.
