# 1. Make Sure Anydesk is installed correctly on the computer:
- When you first download Anydesk it first runs in portable mode (and so it isn't running in the back ground)
- If it is installed correctly on the computer then you should not see the "install anydesk" tile highlighted below
## 2. Set up Unattended access
- To effectively provide support, MovingWalls support team that unattended access set up to the device having the issue in order to check on the status of the computer every day. To set up unattended access:
- select “Set password” in the corner menu
- and set the password here
- For urgent support issues set the password to a simple code like m0vingw5lls. We can remove this access once the immediate issue has been resolved but it is critical that the moving walls support team can access the device remotely as soon as possible. In the past we have been provided with passwords that did not work.
