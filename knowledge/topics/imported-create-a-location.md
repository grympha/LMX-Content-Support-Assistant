# Create a Location
## Overview
A Location in MW Content represents where a device is deployed, such as a building, floor, zone, or screen group. Grouping devices by location helps streamline content scheduling, reporting, and device management.
Locations are linked under Networks, and are essential for organizing screens across different physical or logical areas.
## How to Create a Location
- Navigate to:
Setup → Location
- Click Create
- Fill in the required fields:
Name, Country, Region, Time Zone
Zone Name, Subzone Name
Optionally add Tags and Description
- Ensure Status is enabled
- Click Add

## Best Practices
- Use naming standards like COUNTRY_REGION_ZONE_SUBZONE for clarity
(e.g., MY_KL_TowerA_GroundFloor)
- Always select the correct Time Zone to avoid scheduling issues
- Group screens by logical areas that will share the same content playlist
- Add descriptions for easier future reference
