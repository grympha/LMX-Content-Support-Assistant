# Explore Mode

## What is Explore Mode?

Explore Mode allows users to discover and analyze available inventory before creating a campaign.

## Key Features

- Inventory Discovery
    
- Audience Analysis
    
- Reach Estimation
    
- Campaign Planning
    

## Benefits

- Better planning decisions
    
- Audience insights
    
- Inventory comparison
    

## User Workflow

1. Select location.
    
2. Review inventory.
    
3. Analyze audience.
    
4. Build campaign plan.
