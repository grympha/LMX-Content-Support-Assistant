# Conventional Mode

## What is Conventional Mode?

Conventional Mode provides full campaign planning and control.

## Key Features

- Detailed targeting
    
- Advanced budgeting
    
- Inventory customization
    
- Reporting controls
    

## Benefits

- Greater flexibility
    
- Advanced optimization
    

## User Workflow

1. Define campaign objectives.
    
2. Select inventory.
    
3. Configure targeting.
    
4. Configure budget.
    
5. Launch campaign.
