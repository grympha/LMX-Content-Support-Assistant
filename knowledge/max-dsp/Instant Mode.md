# Instant Mode

## What is Instant Mode?

Instant Mode enables quick campaign creation with minimal configuration.

## Key Features

- Fast setup
    
- Automated recommendations
    
- Simplified workflow
    

## Benefits

- Faster campaign launch
    
- Reduced setup complexity
    

## User Workflow

1. Select inventory.
    
2. Upload creative.
    
3. Set budget.
    
4. Launch campaign.
