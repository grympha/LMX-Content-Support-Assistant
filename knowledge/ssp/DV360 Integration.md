# DV360 Integration

## What is DV360?

DV360 (Display & Video 360) is Google's DSP platform.

## Common Use Cases

- Programmatic DOOH Campaigns
    
- Audience Targeting
    
- Campaign Optimization
    

## Common Issues

### Creative Not Delivering

Possible causes:

- Advertiser configuration missing
    
- Creative approval issue
    
- Inventory eligibility issue
    

### No Impressions

Possible causes:

- Budget restrictions
    
- Inventory unavailable
    
- Targeting too restrictive
