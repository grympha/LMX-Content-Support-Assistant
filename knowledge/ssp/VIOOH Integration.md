# VIOOH Integration

## What is VIOOH?

VIOOH is a global DOOH SSP platform.

## Purpose

Provides access to programmatic DOOH inventory.

## Common Issues

### No Delivery

Possible causes:

- Inventory unavailable
    
- Deal inactive
    
- Creative issue
    

### Creative Errors

Possible causes:

- Unsupported creative format
    
- Campaign configuration issue
