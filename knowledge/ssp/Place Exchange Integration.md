# Place Exchange Integration

## What is Place Exchange?

Place Exchange is a programmatic DOOH marketplace.

## Purpose

Connects DSP demand with DOOH inventory.

## Common Issues

### Empty Ad Slot

Possible causes:

- No eligible campaign
    
- Inventory unavailable
    
- Bid not won
    

### Filler Content Displayed

Possible causes:

- No programmatic content available
    
- SSP fallback activated
