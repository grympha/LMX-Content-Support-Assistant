# Pull To Content Deployment Checklist

## Requirements

-  Windows 1.0.34+ or Android 2.9.1.2+
    
-  Device paired
    
-  Inventory created
    
-  Pull To Content enabled
    

## Verification

-  Inventory name verified
    
-  Content assigned
    
-  Device online
    
-  Content deployed
    

## Validation

-  Content visible on screen
    
-  Playlogs generated
    
-  Inventory correctly mapped
