# New Device Deployment Checklist

## Device Preparation

-  Device powered on
    
-  Internet connection available
    
-  Date and time correct
    
-  Sufficient storage available
    

## LMX Content Installation

-  Latest player installed
    
-  Application launched successfully
    
-  Verification code generated
    

## CMS Configuration

-  Network created
    
-  Location created
    
-  Device added
    
-  Device paired successfully
    

## Content Deployment

-  Playlist created
    
-  Layout created
    
-  Schedule assigned
    
-  Deployment completed
    

## Validation

-  Content visible
    
-  Device online
    
-  Playlogs generated
    
-  Screenshot received
