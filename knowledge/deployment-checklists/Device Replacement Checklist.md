# Device Replacement Checklist

## New Device

-  Device prepared
    
-  LMX Content installed
    
-  Verification code generated
    

## CMS

-  Existing device identified
    
-  New device paired
    
-  Inventory reassigned
    

## Validation

-  Device online
    
-  Content visible
    
-  Playlogs generated
