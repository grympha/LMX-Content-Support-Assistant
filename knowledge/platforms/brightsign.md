# BrightSign Platform Requirements

## Supported Models

- HS123
- XT1143
- HD224

## Processor

- ARM Cortex-A15, Quad-Core 1.0 and 2.0 GHz

## Memory & Storage

- 4 GB RAM / 16 GB ROM
- 2 GB RAM / 8 GB ROM

## System Type

64-bit recommended.

## Supported Media Formats

- Images: PNG, JPG, JPEG
- Videos: MP4
