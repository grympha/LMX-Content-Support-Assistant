# Windows Platform Requirements

## Operating System

Windows 10 and Windows 11. Windows 7 is no longer supported.

## Processor

- Intel Core i5 or i7 recommended
- Intel Pentium II/III or AMD Processor supported

## Memory & Storage

| Tier | RAM | Graphics |
|---|---|---|
| Recommended | 8 GB+ | 1 GB+ |
| Minimum | 4 GB | 1 GB |

## System Type

64-bit recommended; 32-bit supported.

## Supported Media Formats

- Images: PNG, JPG, JPEG
- Videos: MP4, MOV, WEBM, WMV

## Programmatic Playback

Windows 10 and 11 fully support: VAST, URL, HTML, Offline ZIP, and Online ZIP.
