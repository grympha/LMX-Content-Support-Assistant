# Linux Platform Requirements

## Operating System

Ubuntu 18.04 LTS and above.

## Processor

- Intel Core i5 or i7 recommended
- Intel Pentium II/III supported

## Memory & Storage

| Tier | RAM | Graphics |
|---|---|---|
| Recommended | 8 GB+ | 1 GB+ |
| Minimum | 4 GB | 1 GB |

## System Type

64-bit recommended; 32-bit supported.

## Supported Media Formats

- Images: PNG, JPG, JPEG
- Videos: MP4, MOV, WEBM
