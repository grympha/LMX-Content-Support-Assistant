# Android Platform Requirements

## Operating System

Android 11 and above (recommended).

## Processor

- Rockchip RK3328 Cortex A53, Quad Core
- Amlogic S905 Cortex A53, Quad Core

## Memory & Storage

| Tier | RAM | ROM |
|---|---|---|
| Recommended | 8 GB | 128 GB |
| Minimum | 4 GB | 64 GB |

## System Type

64-bit recommended.

## Supported Media Formats

- Images: PNG, JPG, JPEG
- Videos: MP4, MKV

## Programmatic Playback

- Requires Android 11+ with WebView Version 100 or above.
- Supported formats: VAST, URL, HTML online content.
- Outdated WebView may cause black screen or failed rendering.
