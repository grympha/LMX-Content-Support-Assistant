# Supported Formats

## What media formats are supported?

LMX Content supports the following common media formats:

### Images

- JPG
    
- JPEG
    
- PNG
    
- GIF (where supported)
    

### Videos

- MP4 (Recommended)
    
- MOV
    
- WEBM (depending on device capability)
    

### HTML Content

- URL
    
- HTML Package (ZIP)
    

### Programmatic

- VAST
    
- URL-based creatives
    

## Recommendations

For best performance:

- Use MP4 (H.264 codec)
    
- Keep file sizes optimized
    
- Match content resolution to screen resolution
    
- Test content before large-scale deployment
