# Change Device Orientation

## Quick Answer

Device orientation can only be configured during the initial Android player setup, before the device is paired with CMS.

Once a device has been paired, orientation cannot be changed from the player application or the CMS platform without reinstallation and re-pairing.

## For New Android Device Installation

1. Install the LMX Content application.
2. Launch the application.
3. Before entering the verification code, select the desired orientation:
    - Landscape (Horizontal)
    - Portrait (Vertical)
4. Save the configuration.
5. Enter the verification code and complete pairing.

## To Change Orientation After Pairing

Because orientation cannot be adjusted post-pairing through the app or CMS, a new device registration is required.

1. Create a new device in CMS.
2. Generate a new verification code.
3. Uninstall the existing LMX Content application from the device.
4. Reinstall the application.
5. Launch the application.
6. Select the desired orientation before entering the verification code.
7. Pair using the new verification code.

## Recommendations

- Confirm the required screen orientation before starting installation.
- Confirm whether the display will be used in Portrait or Landscape mode.
- Configure orientation before entering the verification code — it cannot be changed afterwards without reinstallation.

## Common Issues

### Orientation option not visible

- Ensure the device has not been paired yet.
- Reinstall the application if the device was previously paired and the option is no longer shown.

### Need to switch from Landscape to Portrait

- Create a new device in CMS.
- Generate a new verification code.
- Reinstall the application and select the new orientation before pairing.

### Content appears stretched or rotated

- Verify content resolution matches the selected orientation (e.g. portrait content requires portrait resolution).
- Verify the display is physically mounted in the correct orientation.

## Future Enhancement

The development team is working on an enhancement that will allow device orientation to be changed directly from the CMS platform without requiring reinstallation or re-pairing.
