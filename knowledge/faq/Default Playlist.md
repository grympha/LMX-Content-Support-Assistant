# Default Playlist

## What is the purpose of Default Playlist?

The Default Playlist acts as fallback content when no active schedule is available.

## Common Use Cases

- Screen startup content
    
- Emergency backup content
    
- Filler content
    
- House advertisements
    

## Recommendations

Always assign at least one item to the Default Playlist.

This helps prevent black screens when schedules expire or fail.
