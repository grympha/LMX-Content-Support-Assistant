# Device Pairing

## How do I pair a device?

### Step 1

Install LMX Content.

### Step 2

Launch the application.

### Step 3

Record the verification code displayed.

### Step 4

Access CMS and add the device.

### Step 5

Enter the verification code.

### Step 6

Verify device appears online.

## Troubleshooting

If pairing fails:

1. Verify internet connection.
    
2. Verify verification code is correct.
    
3. Restart application.
    
4. Retry pairing process.
