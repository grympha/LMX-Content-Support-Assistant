# Device Replacement

## How do I replace a device?

A replacement device can be paired to an existing inventory or location without recreating schedules.

## Procedure

1. Install LMX Content on the new device.
    
2. Generate the verification code.
    
3. Access CMS.
    
4. Locate the existing device.
    
5. Replace or reassign the device.
    
6. Verify deployment status.
    

## Recommendations

- Confirm screen resolution matches the original device.
    
- Verify content is successfully deployed after replacement.
    
- Remove unused devices if no longer required.
