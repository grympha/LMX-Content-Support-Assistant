# Playlogs

## What are Playlogs?

Playlogs record content playback activity on devices.

## Available Reports

### General Playlogs

- Up to 30 days including the current day
    

### Device Playlogs

- Historical playback records
    
- Usually available up to the previous day
    

## Common Uses

- Proof of Play
    
- Campaign verification
    
- Troubleshooting
    
- Playback auditing
    

## Recommendations

Always verify:

- Device Name
    
- Playlist
    
- Content Name
    
- Playback Time
    

when investigating playback issues.

## Campaign Showing in Playlog but Not on Screen

Playlog entries confirm the **player recorded playback** — the campaign IS running on the player. If content is not visible on the physical screen, the issue is with the display output, not the campaign.

### Common causes

- Screen powered off or in standby
- Incorrect display input or source selected (HDMI 1 / HDMI 2 / DisplayPort)
- HDMI cable disconnected or faulty
- LED processor or controller not receiving signal
- Layout resolution does not match screen resolution
- Screen orientation mismatch

### Troubleshooting steps

1. Verify the screen is powered on.
2. Check the display input source selection.
3. Inspect HDMI and power cables.
4. Check LED processor signal and configuration.
5. Verify layout resolution matches the physical screen.
6. Verify screen orientation setting.

Do not conclude the campaign is not playing if playlog entries exist. Investigate the physical display output separately.
