# Pull To Content

## What version supports Pull To Content?

### Windows

Windows 1.0.34 and above

### Android

Android 2.9.1.2 Native and above

## Requirements

1. Device must be paired successfully.
    
2. Inventory must exist in LMX.
    
3. Pull To Content feature must be enabled.
    
4. Inventory name must match the assigned content.
    

## Troubleshooting

If Pull To Content is not working:

1. Verify device pairing.
    
2. Verify inventory mapping.
    
3. Verify content assignment.
    
4. Confirm supported player version.
